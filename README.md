# Woosub Shin Portfolio

Personal portfolio website for quant research, financial econometrics, and research infrastructure roles.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Validation

```bash
npm run lint
npm run build
```

## Boundary

The BTC Futures Research Assistant content is research-only. It is not financial advice, not a trading signal service, and does not imply live trading or paper trading approval.
